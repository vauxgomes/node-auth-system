## Executando o Script de Setup

#### Unix

```sh
chmod +x setup.sh
```

#### Windows

```ps1
./setup.ps1
```

## Variáveis de Ambiente

```
# .env

# Variável de Ambiente para a Porta
PORT=3333

# Segredo do JWT (Deve ser uma string longa e complexa)
AUTH_SECRET="sua_chave_secreta_jwt_muito_longa_e_aleatoria"
```
